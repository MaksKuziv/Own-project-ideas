Історія розвитку Web-розробки

Максим Кузів

  • 1991: Перший веб-сайт

    Творець Всесвітньої павутини (WWW) Тім Бернерс-Лі запустив перший у світі веб-сайт.

  • 1991: документ під назвою HTML-теги

    Тім Бернерс-Лі опублікував документ під назвою HTML-теги . Документ містив опис перших 18 тегів HTML.

  • 1991: HTTP v0.9

    Протокол передачі гіпертексту (HTTP) — це Інтернет-протокол, призначений для обміну гіпертекстовими документами між сервером і веб-браузером.

  • 1991: Перший веб-сервер у США

    Пол Кунц запустив перший у США веб-сервер у Стенфордському центрі лінійних прискорювачів (SLAC).

  • 1993: Перша версія HTML

    Тім Бернерс-Лі написав першу версію HTML. Він був фізиком і комп’ютерним науковцем, який реалізував перший успішний зв’язок між клієнтом протоколу передачі гіпертексту (HTTP).

  • 1994: W3C.org

    Тім Бернерс-Лі заснував міжнародну організацію під назвою World Wide Web Consortium (W3C).

  • 1994: CSS

    Хокон Віум Лі опублікував перший проект каскадних таблиць стилів HTML ( CSS ).

  • 1995: PHP 1.0

    Датсько-канадський програміст Расмус Лердорф розробив першу версію популярної мови сценаріїв PHP 1.0.

  • 1995: HTML 2.0

    Організація IETF опублікувала специфікацію RFC 1866 для HTML 2.0 .

  • 1995: JavaScript 1.0

    Брендан Айх з Netscape розробив першу версію об’єктно-орієнтованого JavaScript, який став широко використовуватися для створення інтерактивних веб-сайтів.

  • 1996: JScript 1.0

    Microsoft впровадила JavaScript в Internet Explorer 3.0 під назвою JScript 1.0.

  • 1996: CSS 1

    W3C видав офіційні рекомендації щодо специфікації мови, каскадних таблиць стилів, рівень 1 (CSS1).

  • 1997: HTML 3.2

    W3C опублікував остаточну рекомендацію, вказуючи мову розмітки HTML 3.2.

  • 1997: Перевірка HTML W3C

    W3C оголосила про запуск W3C HTML Validator.

  • 1997: HTML 4.0

    W3C опублікував офіційні рекомендації щодо специфікації HTML 4.0.

  • 1998: CSS2

    W3C видав офіційні рекомендації щодо специфікації CSS2.

  • 1999: Веб 2.0

    Дарсі Дінуччі опублікував статтю під назвою «Фрагментоване майбутнє» в журналі Print, в якій вперше було введено термін Web 2.0.

  • 2000: PHP 4.0

    Пара ізраїльських програмістів, Зеєв Сураскі та Енді Гутманс, повністю переписали інтерпретатор, який використовується в PHP, і створили версію PHP 4.0

  • 2003: WordPress 0.7

    Веб-розробники Метт Малленвег і Майк Літтл створили першу версію системи керування контентом з відкритим кодом під назвою WordPress 0.7.

  • 2005: Git

    Git - це безкоштовне програмне забезпечення з відкритим кодом для розподіленого контролю версій: відстеження змін у будь-якому наборі файлів.

  • 2005: Ruby

    У 2005 році Ruby став ще однією висхідною зіркою серверної гри, точніше Ruby On Rails ( RoR ), створеної Девідом Хайнемайєром Ганссоном .

  • 2005: DJango

    Приблизно в цей же час, у 2005 році, набула популярності нова платформа Django. Це сприяло використанню Python як серверної мови.

  • 2006: jQuery

    Інженер-програміст Джон Ресіг оголосив про створення бібліотеки JavaScript jQuery на BarCamp NYC.

  • 2006: Sass 0.1.0

    Хемптон Кетлін і Наталі Вейзенбаум розробили препроцесор CSS під назвою Sass 0.1.0 (синтаксично чудові таблиці стилів).

  • 2007: Сітка CSS

    W3C випустив першу пропозицію специфікації CSS Grid.

  • 2008: GitHub

    GitHub - це Інтернет-сервіс для розробки програмного забезпечення та контролю версій за допомогою Git.

  • 2009: NodeJS

    Райан Дал створив щось, що створило б ефект хвилі нових розробників. Він створив спосіб написання JavaScript, який мав би виконувати його як серверну мову. Ця мова програмування відома як NodeJS , і вона швидко стала популярною та широко використовуваною.

  • 2009: Гнучкий макет CSS

    W3C випустив першу пропозицію щодо специфікації CSS Flexible Box layout (Flexbox).

  • 2009: Less

    Less (іноді стилізується як LESS) — це динамічна мова стилів, яка інтерпретується в каскадні таблиці стилів (CSS).

  • 2010: Адаптивний веб-дизайн

    Веб-дизайнер Ітан Маркотт опублікував статтю під назвою «Адаптивний веб-дизайн» в онлайн-журналі A List Apart.

  • 2011: HTML5

    Нарешті, це приводить нас до HTML5, який був випущений у 2011 році .

  • 2011: Bootstrap 1

    Веб-розробники Марк Отто та Джейкоб Торнтон із Twitter почали створювати CSS-фреймворк Bootstrap у середині 2011 року.

  • 2011: Laravel

    У 2011 році Тейлор Отвелл випустив новий фреймворк під назвою Laravel , популярність якого різко зросла протягом наступних кількох років

  • 2012: TypeScript

    TypeScript — мова програмування, представлена Microsoft восени 2012; позиціонується як засіб розробки вебзастосунків, що розширює можливості JavaScript

  • 2013: Bootstrap 3

    Команда розробників з GitHub опублікувала CSS фреймворк Boostrap 3.

  • 2013: React

    React — відкрита JavaScript бібліотека для створення інтерфейсів користувача, яка покликана вирішувати проблеми часткового оновлення вмісту вебсторінки, з якими стикаються в розробці односторінкових застосунків.

  • 2014: Vue.js

    Vue.js — JavaScript-фреймворк, що використовує шаблон MVVM для створення інтерфейсів користувача на основі моделей даних, через реактивне зв'язування даних.

  • 2015: ES6/ES2015

    Шосте видання, ECMAScript 6 ( ES6 ), пізніше перейменоване на ECMAScript 2015, було завершено в червні 2015 року.

  • 2015: React Native

    React Native — це програма інтерфейсу користувача з відкритим вихідним кодом, створена Meta Platforms, Inc.

  • 2016: Angular

    Angular — написаний на TypeScript front-end фреймворк з відкритим кодом, який розробляється під керівництвом Angular Team

  • 2017: Gulp

    Gulp — таск-менеджер для автоматичного виконання завдань (наприклад, мініфікаціі, тестування, об'єднання файлів), написаний на мові програмування JavaScript.
